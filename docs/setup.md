## 1. Smart Contract

```bash
cd onchain
aiken build        # compiles в†’ plutus.json
aiken check        # run unit tests (if any)
```

The compiled CBOR in `plutus.json` is already embedded in `offchain/contract.ts` and `frontend/app/lib/contract.ts`. Only re-embed if you modify the validator.

## 2. Environment Variables

Create `.env` in the project root (for offchain tests):

```env
BLOCKFROST_API_KEY=previewXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
ALICE_MNEMONIC=word1 word2 word3 ... word24
BOB_MNEMONIC=word1 word2 word3 ... word24
CHARLE_MNEMONIC=word1 word2 word3 ... word24
```

Get a free Blockfrost API key at https://blockfrost.io вЂ” select **Preview** network.

For the frontend, create `frontend/.env.local`:

```env
NEXT_PUBLIC_BLOCKFROST_API_KEY=previewXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

## 3. Offchain (Headless Tests)

```bash
cd offchain
npm install
node --loader ts-node/esm test.ts
```

The test script runs two scenarios:
- **Scenario 1:** Alice creates campaign в†’ Bob donates в†’ Charlie withdraws (as soon as goal is met, no deadline needed)
- **Scenario 2:** Alice creates campaign в†’ Bob donates в†’ Bob reclaims (after deadline, goal NOT met)

Wallets must be funded with at least 20 tADA each on Preview testnet.
Get tADA from https://docs.cardano.org/cardano-testnets/tools/faucet/

## 4. Frontend

```bash
cd frontend
npm install
npm run dev     # development server at http://localhost:3000
npm run build   # production build
```

### Wallet Setup

Install a Cardano browser wallet (Eternl recommended for testnet):
1. Install [Eternl](https://eternl.io) extension
2. Switch network to **Preview Testnet**
3. Fund with tADA from the faucet
4. Set collateral (5 ADA) in wallet settings вЂ” required for script transactions

### Using the dApp

1. Open http://localhost:3000
2. Click the wallet button in the header to connect
3. **Create Campaign:** Fill in beneficiary address, goal ADA, deadline, initial donation
4. **Donate:** Click any active campaign card, enter amount, click Donate
5. **Withdraw:** Available to beneficiary as soon as goal is met (no deadline required)
6. **Reclaim:** Available to contributors after deadline if goal is NOT met

## 5. Rebuilding Contract Artifacts

If you modify `crowdfund.ak`:

```bash
cd onchain
aiken build
```

Then copy the new `compiledCode` from `onchain/plutus.json` into:
- `offchain/contract.ts` в†’ `SCRIPT_CBOR_SINGLE` constant
- `frontend/app/lib/contract.ts` в†’ `SCRIPT_CBOR_SINGLE` constant

And update `SCRIPT_HASH` with the new hash from `plutus.json`.

## Troubleshooting

**"No collateral UTxO"** вЂ” Set 5 ADA as collateral in your wallet settings (Eternl: Settings в†’ Collateral)

**"Module not found: @meshsdk/..."** вЂ” Run `npm install` in the relevant directory

**Transaction submission fails** вЂ” Check Blockfrost dashboard for error details; ensure wallet is on Preview testnet
